# 🦡 AuthoryProf (오소리) - 교과목 프로파일 자동화 솔루션

**"지겨운 '복붙' 노가다, 이제 AI에게 맡기세요."**

AuthoryProf(애칭: 오소리)는 교육과정 개발 시 가장 번거로운 작업 중 하나인 **'교과목 프로파일 작성'을 자동화**해주는 데스크톱 애플리케이션입니다. 기존의 한글(HWP) 보고서를 분석하여 데이터를 추출하고, Google Gemini AI를 활용해 새로운 양식에 맞춰 자동으로 문서를 생성합니다.

---

## ✨ 주요 기능 (Key Features)

* **🔄 역설계 (Reverse Engineering):** 기존 교육과정 개발 보고서(HWP)의 내용을 AI가 분석하여 교과목 프로파일 데이터(JSON)로 자동 변환합니다.
* **📄 문서 자동 생성:** 추출된 데이터를 바탕으로 지정된 템플릿(HWP)에 내용을 자동으로 입력하여 결과물을 생성합니다.
* **🔗 자동 병합:** 생성된 개별 교과목 프로파일 파일들을 하나의 HWP 파일(`Merged_Report.hwp`)로 깔끔하게 병합합니다.
* **🤖 AI 기반 분석:** Google Gemini API(Flash/Pro 모델 지원)를 연동하여 단순 텍스트 추출을 넘어 맥락에 맞는 데이터를 생성합니다.
* **✅ 데이터 검증:** 필수 항목 누락 여부를 사전에 체크하여 문서의 완성도를 높입니다.
* **🎨 미니멀 UI:** 복잡한 설정을 배제하고 꼭 필요한 기능만 담은 직관적인 플랫(Flat) 디자인을 적용했습니다.
* **🚀 자동 업데이트:** GitHub와 연동되어 최신 버전이 출시되면 자동으로 감지하고 알림을 제공합니다.

## 🛠️ 설치 및 준비 (Prerequisites)

이 프로그램은 **Windows 환경**과 **한글(Hancom Office)** 프로그램이 설치된 환경에서 작동합니다.

1. **필수 소프트웨어:** 한글(HWP) 2018 이상 권장
2. **Google AI Studio API Key:** [Google AI Studio](https://aistudio.google.com/)에서 무료로 발급받을 수 있습니다.
     (무료 API는 토큰사용에 따라 정지 될수 있으니 신용카드 입력 후 후불 종량제를 사용해 보시기 바랍니다.)
---

## 📖 사용 방법 (How to Use)

1. **실행:** `AuthoryProf_v1.0.3.exe`를 실행합니다.
* 최초 실행 시 `내 문서 > AuthoryProf_Works` 폴더가 자동으로 생성됩니다.


2. **보고서 준비:** `AuthoryProf_Works > Report_hwp` 폴더 안에 기존 교육과정 보고서(HWP)를 넣습니다.
3. **API 설정:** 우측 상단 **[API]** 버튼을 눌러 발급받은 Gemini API 키를 입력합니다.
4. **역설계 실행:** **[역설계 실행]** 버튼을 클릭하면 AI가 문서를 분석하여 `input_json` 폴더에 데이터 파일을 생성합니다.
5. **문서 생성:** **[문서 생성 시작]** 버튼을 누르면 JSON 데이터를 HWP로 변환하고 병합합니다.
6. **결과 확인:** `merge_hwp` 폴더에서 완성된 `Merged_Report.hwp` 파일을 확인합니다.

---

## 📂 폴더 구조 (Directory Structure)

프로그램 구동 시 `Documents/AuthoryProf_Works` 내부에 다음과 같은 구조가 생성됩니다.

```text
AuthoryProf_Works/
├── 📂 Report_hwp/      # (입력) 원본 교육과정 보고서를 넣는 곳
├── 📂 input_json/      # (중간) AI가 생성한 데이터 파일이 저장되는 곳
├── 📂 output_hwp/      # (중간) 개별 생성된 HWP 파일들이 저장되는 곳
├── 📂 merge_hwp/       # (결과) 최종 병합된 파일이 저장되는 곳
├── 📄 config.ini       # 사용자 설정 파일 (API 키 등)
└── 📄 template.hwp     # 프로파일 양식 템플릿

```
