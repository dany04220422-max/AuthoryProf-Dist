# AuthoryProf-Dist
AI사무자동화 툴 AuthoyProf의 공식 배포 및 업데이트 공개 채널입니다. AI사무자동화를 실현하여 인간의 'Work-Life Balance' 을 지킵니다.

# AuthoryProf (Official Release)

이 저장소는 **AuthoryProf** 프로그램의 설치 파일(.exe)과 업데이트 정보를 제공하는 공간입니다.

## 📥 다운로드 및 설치 (Download)
오른쪽 사이드바의 **[Releases]** 섹션에서 최신 버전(`Setup.exe`)을 다운로드하세요.

## ⚠️ 참고 (Note)
* 이 저장소는 소스 코드를 포함하지 않습니다. (Binary Distribution Only)
* 프로그램 실행 시 이곳의 정보를 확인하여 자동으로 업데이트를 알립니다.
